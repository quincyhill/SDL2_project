# SDL2 Project
## Just a little things to mess with c++ and sdl2 and to learn makefiles

## Dependencies 
* SDL2
* lld
* cmake
* g++
* git


## Clone Source Code
git clone https://github.com/quincyhill/SDL2_project.git

## Build Environment
Builds the game demo
`$ make`

Deletes any object files if you wish
`$ make clean`

## Run
`$ ./game`
